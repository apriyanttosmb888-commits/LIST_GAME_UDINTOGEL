🔧 KONFIGURASI FIREBASE DIPERLUKAN

Ikuti langkah berikut untuk menghubungkan Firebase:

  1. Buka console.firebase.google.com
  2. Buat project baru → aktifkan Firestore Database
  3. Di Rules Firestore, set: allow read, write: if true;
  4. Pergi ke Project Settings → Web App → copy config
  5. Ganti nilai FIREBASE_CONFIG di bawah dengan config kamu
  6. Upload ulang file ini ke GitHub Pages
